# Homebrew Tap for Bagdock CLI

Install the [Bagdock CLI](https://github.com/bagdock/bagdock-cli) via Homebrew.

## Install

```bash
brew install bagdock/cli/bagdock
```

## Update

```bash
brew upgrade bagdock/cli/bagdock
```

## Uninstall

```bash
brew uninstall bagdock
brew untap bagdock/cli
```

## About

This tap installs `@bagdock/cli` from npm via Node.js. It requires Node.js 20+.

For alternative install methods, see the [CLI README](https://github.com/bagdock/bagdock-cli#install).
